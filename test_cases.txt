# Тестовые кейсы из публичной выборки LocalScript

## Задача 1: Проверка email
Напиши функцию is_valid_email(email), которая проверяет, является ли строка корректным email адресом.
Требования:
- Должен содержать символ @
- Должен содержать точку после @
- Не должен начинаться с @ или точки
- Не должен заканчиваться на @

## Задача 2: Числа Фибоначчи
Напиши функцию fibonacci(n), которая возвращает n-ое число Фибоначчи.
Используй мемоизацию для оптимизации.

## Задача 3: Палиндром
Напиши функцию is_palindrome(str), которая проверяет, является ли строка палиндромом.
Игнорируй пробелы и регистр.

## Задача 4: Факториал
Напиши рекурсивную функцию factorial(n), которая вычисляет факториал числа n.

## Задача 5: Сортировка пузырьком
Напиши функцию bubble_sort(arr), которая сортирует массив методом пузырьковой сортировки.

## Задача 6: Поиск максимума
Напиши функцию find_max(arr), которая находит максимальный элемент в массиве.

## Задача 7: Реверс строки
Напиши функцию reverse_string(str), которая переворачивает строку.

## Задача 8: Подсчёт слов
Напиши функцию count_words(text), которая подсчитывает количество слов в тексте.

## Задача 9: Простые числа
Напиши функцию is_prime(n), которая проверяет, является ли число простым.

## Задача 10: Сумма массива
Напиши функцию sum_array(arr), которая вычисляет сумму всех элементов массива.
