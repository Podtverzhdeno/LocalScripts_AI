# МТС True Tech Hack 2026 - Демо-сценарии

Этот каталог содержит 5 бизнес-сценариев для демонстрации возможностей LocalScript на хакатоне МТС.

## 📋 Список сценариев

### 1. Email Validator (`01_email_validator.txt`)
**Задача:** Валидация email адресов по RFC 5322  
**Применение:** Формы регистрации, проверка данных, санитизация входных данных  
**Сложность:** ⭐⭐☆☆☆

**Демонстрирует:**
- Работу с регулярными выражениями
- Валидацию бизнес-правил
- Обработку граничных случаев

---

### 2. Kafka Integration (`02_kafka_integration.txt`)
**Задача:** Форматирование данных для отправки в Kafka  
**Применение:** Event streaming, микросервисная архитектура, интеграция систем  
**Сложность:** ⭐⭐⭐☆☆

**Демонстрирует:**
- Нормализацию данных
- Создание структур сообщений
- Сериализацию в JSON
- Работу с временными метками

**Пример из речи Александра Бордаша:**
> "Я хочу проинтегрировать свой бизнес, там, например, номер телефона, и хочу его положить в Kafka"

---

### 3. REST API Client (`03_api_wrapper.txt`)
**Задача:** Создание клиента для REST API пользовательского сервиса  
**Применение:** Интеграция с внешними API, микросервисы, автоматизация  
**Сложность:** ⭐⭐⭐☆☆

**Демонстрирует:**
- CRUD операции
- Построение HTTP запросов
- Query string параметры
- Объектно-ориентированный подход

---

### 4. Data Transformer (`04_data_transformer.txt`)
**Задача:** Трансформация JSON между форматами разных API  
**Применение:** Интеграция систем, миграция данных, адаптеры  
**Сложность:** ⭐⭐⭐⭐☆

**Демонстрирует:**
- Парсинг JSON
- Маппинг полей
- Работа с вложенными структурами
- Flatten/nest операции

---

### 5. Event Processor (`05_event_processor.txt`)
**Задача:** Обработка событий из очереди и сохранение в БД  
**Применение:** Event-driven архитектура, обработка потоков данных  
**Сложность:** ⭐⭐⭐⭐⭐

**Демонстрирует:**
- Парсинг Kafka сообщений
- Генерацию SQL запросов
- Batch processing
- Error handling
- Комбинацию нескольких паттернов

---

## 🚀 Как использовать

### Через CLI:
```bash
python main.py --task-file examples/mts_hackathon/01_email_validator.txt
```

### Через Web UI:
1. Откройте http://localhost:8000
2. Скопируйте содержимое файла сценария
3. Вставьте в поле "Task Description"
4. Нажмите "Generate"

### Через API:
```bash
curl -X POST http://localhost:8000/api/run-task \
  -H "Content-Type: application/json" \
  -d '{"task": "содержимое файла сценария"}'
```
